# My Dashboard

A simple personal dashboard built in Python with PyQt6, includes some handy modules. Made for me.

## Current functionality

- **System Stats Panel**
  - Shows CPU usage
  - Shows RAM usage
  - Shows Disk usage
  - Shows Network usage

- **Other Modules - not yet implemented**
  - To-Do Panel
  - Weather Panel
